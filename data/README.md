## Data Source

This project uses the NY DFS Automobile Insurance Complaint Rankings dataset,
publicly available via the NY Open Data portal.

**Direct API URL (used by the notebook — no manual download needed):**
https://data.ny.gov/api/views/h2wd-9xfe/rows.csv?accessType=DOWNLOAD

**Dataset page:**
https://data.ny.gov/Government-Finance/Automobile-Insurance-Company-Complaint-Rankings-Be/h2wd-9xfe

The dataset covers 2009–2024, contains 2,461 records across 243 unique
insurers, and is updated annually by the NY Department of Financial Services.
No local copy of the data is stored in this repository.
